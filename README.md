# Citadel Systems IT - Corporate Website

A professional, fully-functional corporate IT services website with a sophisticated dark theme and enterprise-grade features.

## Features

- **Modern Dark Theme**: Sophisticated navy/charcoal with cyan/blue accents
- **Fully Responsive**: Works perfectly on all device sizes
- **Animated Counters**: Statistics that animate when scrolled into view
- **Interactive Dashboard**: Real-time metrics display
- **Tabbed Solutions Interface**: Industry-specific solutions with smooth transitions
- **Functional Contact Form**: With validation and submission feedback
- **Case Studies Section**: Enterprise success stories
- **Technology Partners Showcase**: Partner logos grid
- **Progress Bars**: Animated progress indicators for metrics

## Technical Specifications

- **HTML5**: Semantic, accessible markup
- **CSS3**: Modern CSS with custom properties (variables), Flexbox, and Grid
- **JavaScript (ES6)**: Vanilla JS with no dependencies
- **Fonts**: Inter from Google Fonts
- **Icons**: Font Awesome 6
- **Form Handling**: Client-side validation with simulated backend integration

## Color Scheme

- Primary Background: `#0a192f` (navy)
- Secondary Background: `#112240` (lighter navy)
- Text Primary: `#e6f1ff` (off-white)
- Text Secondary: `#8892b0` (muted slate)
- Accent: `#64ffda` (cyan)
- Primary Button: `#4d7cfe` (bright blue)
